# 阶段⑥：功能真值表（当前实现基线）

本文档用于记录 `TgGroupRobot` 当前版本的真实功能状态，作为后续补全、回归测试和 README 对账基线。

状态定义：

- `已闭环`：管理员或用户可直接进入真实流程，主链路可用，且已有回归测试覆盖
- `已有能力，待扩展`：主链路可用，但在深度能力、批量操作或更细化产品层面仍可继续增强
- `暂时关闭`：相关底座保留，但当前版本不对用户形成付费限制或购买流程
- `仍待实现`：代码或菜单中暂无完整产品能力，不应在对外说明中写成已完整支持

## 11. 维护约定

  - `README.md`

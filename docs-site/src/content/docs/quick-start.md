## 管理员入口

1. 把机器人加入目标群组，并授予需要的管理员权限。
2. 在私聊发送 `/start`，点击管理入口。
3. 如果管理多个群组，先通过按钮选择要配置的群。
4. 从按钮菜单进入对应页面，按步骤完成配置。

## 推荐配置顺序

1. 先跑一次“群组健康检查”，确认机器人权限。
2. 开启“新人验证”“强制关注”或“新成员限制”，建立入群保护。
3. 配置“反垃圾”“防刷屏”“违禁词”和“惩罚策略”。
4. 再配置积分、邀请、定时消息、轮播广告和活动类功能。

## 文档使用方式

每个功能页都包含分步教程、Telegram 风格操作图和常见问题。配置时按页面里的步骤走，遇到异常先确认前置条件、操作顺序和 Q&A。
